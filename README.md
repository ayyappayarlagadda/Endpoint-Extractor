# Endpoint Extractor

## 📖 Description

The Endpoint Extractor is a powerful Chrome extension designed to help developers, security researchers, and QA engineers easily extract and analyze API endpoints directly from any web interface. By observing the web application's traffic and structure, this tool maps out the application's routing and provides a clean, organized view of the available endpoints.

## ✨ Features

 - Advanced Search: Quickly search and filter through extracted endpoints to find exactly what you need. 
 - HTTP Methods Labeling: Visual tags for different HTTP methods (e.g., GET, POST, PUT, DELETE) to easily distinguish the purpose of each endpoint. 
 - JS Filter: Isolate and extract endpoints specifically hidden within JavaScript files to analyze client-side routing.

## 🛠️ Installation (Chrome)
To use this extension, you will need to install it locally using Chrome's "Load unpacked" feature.

**Bash**

    git clone https://github.com/Ayyappachowdary/Endpoint-Extractor.git

 - Open Extensions Page: Open your Google Chrome browser and type the
   following into your URL address bar, then hit Enter: `chrome://extensions/` 
 - Enable Developer Mode: In the top right corner
   of the Extensions page, toggle the Developer mode switch to ON.
 - Load the Extension: Click the Load unpacked button that appears in the top
   left corner. 
 - Select the Folder: Browse your file system and select the directory containing the extension's code (make sure to select the folder that contains the   manifest.json file). 
 - Pin the Extension (Optional): Click the puzzle piece icon in your Chrome toolbar and
   pin the extension for easy access.
